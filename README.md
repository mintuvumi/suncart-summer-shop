
# Project Name: SunCart 

##  Project Overview
## SunCart – Summer Essentials Store


SunCart is a modern summer eCommerce platform where users can explore and purchase seasonal products like sunglasses, summer outfits, skincare, beach accessories, and more.

Users can browse products, view detailed information, and place orders after authentication.

---

##  Purpose

The goal of this project is to build a fully responsive and modern eCommerce web application using Next.js, Tailwind CSS, DaisyUI, and BetterAuth.

##  Key Features

-  Authentication system (Email/Password + Google Login)
-  Protected Product Details Page
-  Product listing with dynamic routing
-  Hero section with summer offers
-  Popular products section
-  User profile page
-  Update profile feature (name & image)
-  Fully responsive (Mobile, Tablet, Desktop)
-  Unique summer-themed UI design
-  Fast performance with Next.js App Router

---

##  Pages & Routes

- → Home Page
- products` → All Products
- products/[id]` → Product Details (Protected)
- login` → Login Page
- register` → Register Page
- my-profile` → User Profile
- update-profile` → Update Profile
